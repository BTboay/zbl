奥地利宰相——梅特涅¶

1818年5月5日马克思出生，而1815年欧洲出现了一个庞大的帝国，横跨半个欧洲，可以称其为大德意志联邦，管理这个帝国的人是奥地利的宰相，梅特涅此时登场了。梅特涅出生于1773年到1859年离世，在奥地利爆发三月革命后，梅特涅辞职离开，当他离开后1848年俾斯麦闪亮登场，梅特涅治理下的大德意志地区和俾斯麦治理下的德意志帝国构成了以德国为中心精彩纷呈的欧洲近代史。

梅特涅是马克思少年时期和青年时期的重要压迫者，他对马克思思想的形成有着重要的影响。梅特涅是德国人，也是当时大德意志地区的实际领导者，梅特涅和俾斯麦相比，他更像中国的苏秦张仪这样的人物，属于当时欧洲的纵横家。梅特涅的夫人艾利诺·考尼克斯是一位女伯爵，梅特涅是靠婚姻从而进入到奥地利的上层社会的，其中有一段梅特涅和拿破仑的故事。拿破仑平息了法国大革命，摄政法国之后，四处攻伐，德国和奥地利都被法国大军占领，有一次拿破仑在维也纳参加一个化妆舞会，个子矮小的拿破仑身穿法国军官的服饰，主动邀请艾利诺·考尼克斯跳舞，一边跳舞一边带着女伯爵向房间走去。当时女伯爵感到愤怒和惊讶，但又不好在大庭广众下发作，当进入到房间后，拿破仑摘下面具吓了女伯爵一跳，原来这么粗俗的人居然是拿破仑，其实拿破仑只是求女伯爵一件私事，他想娶奥地利皇帝的妹妹，他需要一段婚姻。当然女伯爵很生气很无语，回去后将此事告诉了梅特涅，梅特涅开始也很愤怒，但得知拿破仑的实际意图后会心一笑。拿破仑事实上是希望通过婚姻来实现某种政治诉求，本来拿破仑是想娶俄国沙皇的妹妹，但被拒绝，如果奥地利也拒绝，那么奥法战争可能会替代俄法战争，因为这段婚姻的实现，法国和奥地利成为了盟国，在拿破仑进攻俄国时奥地利提供了很多方便。但奥地利有梅特涅这样的政治家在，是不会真心实意的和法国结盟的，奥地利答应拿破仑一起攻打沙俄的三万大军没有从奥地利出发，当拿破仑兵败回来后，奥地利连通英国，俄国和法国其他势力，做掉了拿破仑。

奥匈帝国不是强大的帝国，一直处在欧洲的夹缝中，在特定的历史时期梅特涅作为宰相发挥了重要的作用，这是梅特涅个人才干和能力的表现。梅特涅是一个残酷的保守主义者，他对奥地利和德意志的统治，采取了极为残酷和严厉的手段维护了奥匈帝国和普鲁士的旧贵族和教士的特权利益，当时的英法正在快速走向现代化，但由于梅特涅的存在，普鲁士和奥地利没能尽快走向现代化，但他用犀利的外交手段维持了大德意志地区的阶段性和平。
