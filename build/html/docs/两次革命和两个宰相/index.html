两次革命和两个宰相¶

我们首先需要回到历史，因为马克思生活在19世纪的欧洲，这是一个风云突变精彩纷呈的时代，不讲历史是讲不清楚马克思的，更是讲不了资本论的，但是回到历史的工作量是巨大的，哪怕您通读过《企鹅欧洲史》您对欧洲的这段历史也是流于表面的。本文的核心是资本论，而不是某种大学的历史课，我们希望将这一部分简化一下，将其概括为两次革命与两个宰相，两次革命分别是“工业革命”“法国大革命”，两个宰相分别是"梅特涅““俾斯麦”。

工业革命发端于1750年前后的英国，蒸汽机和珍妮纺纱机的出现，改变了整个欧洲也改变了人类的历史进程。如果说工业革命从物质层面改变了欧洲，那么法国大革命就是从精神层面改变了欧洲，是一次重要的思想解放运动。这两次革命奠定了欧洲基本的政治经济生态，深刻影响了普鲁士也就是后来的德国，在这样的环境下马克思诞生了，马克思及其资本论，以及全部的社会主义或者共产主义学说不是从天上掉下来的，而是在那样一片土壤上生长出来的，换句话说马克思只是演绎了整个的历史进程，即便没有马克思也一定会走向这样一个过程，只不过上帝选择了马克思。

俾斯麦和马克思是同龄人，而梅特涅对少年和青年马克思有极深刻的影响，这两位宰相经历了工业革命和法国大革命，并且对后来大德意志帝国的崛起起到决定性作用，他们从另外的角度让我们认识马克思。德国人是很会写自传的，俾斯麦的《回忆与思考》与希特勒的《我的奋斗》，显然俾斯麦文学功底略显薄弱了，但回忆与思考这本书具有很重要的价值，因为这是在德国崛起的过程中形成的，凝聚着俾斯麦作为最高统治者对整个德意志民族国家的深刻思考，其中的一些想法对今天的中国有着无与伦比的意义。显然俾斯麦和马克思是两个极端，站在国家的角度来看，俾斯麦显然更具价值，但是如果站在人类或者哲学史的角度考虑，马克思无疑是无比辉煌的，但历史是这样构成的，我们不能执着于一端，可以这样讲，没有俾斯麦就没有德意志的现代化。
