商品定价¶

商品定价也就是资产定价理论，我们需要理解的是定价的逻辑，商品是怎么定价的，一件商品为什么值这个价格。通常对商品定价有一个相对的等价模型，如果没有黄金这样的一般等价物，那么我们在进行商品交换的时候，例如用5斤米换1斤肉，米和肉就是相对的一般等价物。一般等价物形成我们对交换价格的一般性理解，从一般价值形成过渡到货币形式，商品定价过程就完成了。商品定价不是基于合理性构建的，而是一种约定俗成，也就是所谓市场，商品定价逻辑就是市场定价逻辑。相对等价模型进行哲学抽象，才能从一般价值形式过度到货币形式，我们对价值是存在基本判断的，在购买或者给别人的时候，我们知道它的使用价值，但真正形成交换价值是偶然的，为什么5斤米换1斤肉？这个公式是怎么计算出来的，我们无法精算每一件商品中凝结的人类劳动，这是一种直觉。货币化或资本化会对商品定价产生逆向影响，持有货币者往往代表需求，会构成对需求价值的定义和构建。

商品定价逻辑基本上由三部分组成：

1.凝结的劳动：是总成本，是底价，是对劳动的补偿。

2.稀缺性：是超过成本的溢价，溢价与稀缺性是正相关的。如果一个东西没人要，价格是会低于成本的。稀缺性可能是真实的稀缺，也可能因为信息不对称或垄断而产生。

3.垄断：垄断是人为造成的某种稀缺性表达。

劳动、稀缺性、垄断共同构成基本定价逻辑，在现代世界随着资本化越来越强大，垄断金融资本的出现，垄断成了当前定价的主要逻辑。

这三个概念可以用稀缺性进行统一表达，劳动的价值性是由于劳动时间和劳动力的有限性造成的，表达为供应上限，垄断是人为造成供应的稀缺。马克思难以完成基于稀缺性的思考，因为在那个时代没有绝对稀缺性，通过增加劳动力和劳动时间，资源供应是会增加的，而现代社会的市场供应是由货币决定的，货币的稀缺性可以被金融资本垄断，造成市场的整体通缩。商品定价应当回归它的本源，也就是凝结劳动和自然稀缺性的价格。

弗里德曼有一句话“通胀是货币现象”，通胀就是涨价，涨价是货币现象。弗里德曼站在现代社会经济体系下，否定了商品或资产定价的劳动价值和自然稀缺性，而将“货币”这种特殊商品的垄断，作为现代商品定价理论的核心。弗里德曼把这个世界看的很透彻，也就是大资本在决定世界的定价，是1945年后世界的广谱性现象。

由于国际化的形成，几乎所有商品特别是大宗商品和重要资源，全部由跨国资本垄断和操纵，现代商品的稀缺性很大程度上是信息不对称的结果。在美股和中国房地产上进行着充分的表达，美国股价并不是表达了那些股票、股权的稀缺性，而是资本过剩下，资本对资产的垄断和操纵，中国房地产也同样表达为资本对部分资产的垄断和操纵，这是一种通货膨胀的逻辑而不是商品定价的逻辑。你手中的货币在不断贬值，换成资产才能保护自己的财产，即使那个资产你并不需要，你需要这个过程来保值、增值，其中还夹杂着不少投机。

商品定价或者资产定价的不合理是因为货币定价的问题，这个不合理或波动的过程形成了对劳动者的剥削，这个剥削远远超过了马克思时代产业资本通过劳动价值的剥削程度。商品定价理论的重要性体现出来了，因为其中存在着剩余价值，也就是劳动价值的剥削。不合理的资产定价构成了我们所谓“割韭菜”的过程。

资本对劳动者的剥削目前看来是有办法解决的，但金融帝国主义对其他国家与民族的压迫，我们未必找到了好的解决方案。我们不断在讨论商品定价权，它在谁的手上？黄金在谁的手上？比如碳排放权这些东西在谁的手上？从石油危机到MMT，从黄金到碳排放，我们是否看到了背后定价权的那只手。从舆论、文化、价值观、市场，我们看到了一个系统性的整体压迫，这是一种阴险的系统性制度编排。
