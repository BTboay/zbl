国家与资本的关系¶

资本不是静态的，用静态的思维看资本，那最多称之为资金。资本是动态的，是流动的，资本的流转可以分为不同的层级进行解读。

  1. 在不同的环节流转

  2. 在不同的产业流转

  3. 在不同的空间流转

马克思在资本论第二卷和第三卷中研究了资本流转的问题，也同时在揭示国家与资本之间应该形成一种怎样的关系，这也是为什么共同富裕的推进表达了我国对资本治理的基本态度，反过来讲，共同富裕所做的就是在处理国家与资本之间的关系。

资本流动貌似是因为其逐利性，因为在不同的空间或者产业，或者某个环节，其资本利得的水平存在差异，而逐利的资本循着资本利得的水平高低，在不同的环节流动，这是一种普遍的看法。但这事实上是一种因果颠倒的说法，要素的流动从来不是市场定价的结果，而是要素流动造就了那样的定价。而要素流动的根本原因是政策导向。政策导向，使得要素向某些环节或空间流动，而要素流动后的积累形成了在该环节和该空间的某种定价逻辑。直到今天能够看明白这个问题的人屈指可数，而马克思十分清楚的认识到了这一点，这就是为什么《资本论》将政治和经济混在一起谈，借助经济问题深挖政治问题，就在于经济是政治的产物，治理经济要从政治出发。

国家与资本的关系，其实就是国家这样一个政治机器，应该制定出怎样的制度来管理资本的流转问题。马克思没有回答这个问题，是因为他看到了国家这个机器在处理资本的时候，会出现问题，可能这些问题他没有思考清楚，也可能他思考到人性的时候彻底的失望了，无论怎样，马克思没有给出一个结论。但革命的烈火已经席卷了整个欧洲，无产阶级迫切需要一个建立无产阶级国家的基本方针，所以列宁匆忙写下《国家与革命》，将国家与资本的关系，用国家资本主义的形式表达了出来。
