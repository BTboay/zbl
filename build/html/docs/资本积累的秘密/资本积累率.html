资本积累率¶

经济增长的源泉是资本积累率，过去几年主流的经济学家都在说，中国的经济体量很大了，所以经济增长速度放缓是正常的，这是无稽之谈。导致中国经济增长放缓的真正因素，在于我们的资本积累率下降了，甚至到了个位数。不妨牢骚几句，当今的政治经济学并非科学，而是一种掺杂了意识形态的某种炼金术，改革开放后受西方意识形态和教育学术的影响，我们对我们自己的发展历史是具有认识偏差的，对西方国家的发展历史同样具有认识的偏差，使得我们的学者们构建的理论和现实相差甚远，万幸今天中国的这一代领导人不是楞头的大学生，而是具有丰富治理经验的政治家。

资本积累率需要稳定在一个合理的水平，太高了不好，因为这压低了我们的生活，太低了不行，这使得我们的再生产能力下降。2008年美国金融风暴之后，中国配合美国进行资本扩张，一直到2015年我们的资本积累率维持在一个高位，而2016年我们明确取消人民币和美元之间的联系汇率制度，而美元又正好进入收缩阶段，这个时候我们出现大规模的资本外逃，资本积累率开始疯狂下降，我们事实的进入了一轮经济萧条中。我们一定要处理好资本积累的问题，要有看法也要有算法，这是治理的一个核心。
