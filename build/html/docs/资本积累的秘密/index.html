资本积累的秘密¶

资本论第一卷的最重要部分就是资本积累，只有说清楚资本积累才能说清楚资本流转，这是一个互相牵扯的结构性问题。资本积累，资本利得，资本流转共同构成资本的动态过程，过去总讲分配的公平性，但是长时间过去都未曾实现，我们应当主动跳出其中，来讨论资本分布的合理性在公平中的重要作用。
