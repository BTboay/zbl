货币的功能与意义¶

人生中有两样东西的分配至关重要，一个是时间，一个是金钱。你的时间是有限的，你的金钱是有限的，如何分配你的时间和金钱将决定你的命运。如果你拥有一万块钱，你会拿它做什么呢？去娱乐消费，大吃一顿，还是去参加补习班学习知识，还是买股票进行投资。这些选择意味着什么呢？这意味这你出于本能进行着生产力工具的确认，第一是恢复再生产，你作为一个劳动力，要恢复到可生产状态需要吃饭、穿衣、睡觉；第二是提高生产力水平，学习新知识新技术就是提高生产力水平；第三是获得资本性收益，通常表现为资金在资本市场的运作。

大多数人在拥有钱之后都面临货币功能的选择，钱是有限的，再生产的恢复可以极其奢华也可以适当就好，这表明一种货币分配的占比问题，你投入再生产多少、投入提高生产力多少、投入资本性收益多少。这些问题在一次性上看似并不重要，但当你持续的进行分配10年、20年后就会发生显著的变化，一生过去就会与他人截然不同。这是一个分布问题。

多数人对钱的理解是一种刹那间的感觉，钱可以兑换所需的物品，可以将此理解为再生产需要，但其中夹杂了太多的欲望、虚荣心和其他的东西，会导致人在再生产部分的分配不够理性，超出正常需求，同时也会对生产工具形成一定的忽视。生产力提高的投入应当是终身的，在人生的不同阶段要提升不同的能力，其中包含无形的认知水平以及有形的实际技能。生产力水平的不断提高是决定个人命运的重要方面，它与资本性收入构成命运的两根支柱。

“钱”是价值尺度，可以衡量一个商品的价值，可以衡量一个资产的价值，但钱不能衡量一个人的价值。人的价值不是由他做了多大的官或者拥有多少财富决定的，而是他的社会性程度，用宗教的语言就是他的“业”。马克思在《资本论》中当然不会谈论人的价值，马克思讨论的是劳动，他将劳动商品化了，马克思讨论的是劳动的价值和商品的价值，而它们的价值是用钱来衡量的。如同刚才的例子，你花钱去消费和你花钱去学习，其分别是商品化想法和资本化想法，是两个不同的路径，最终产生两个不同的结果。

我们要在马克思货币意义的基础上进行延伸，除了一般等价物、价值尺度、价值存储、交易媒介外，还有恢复再生产手段、提高生产力手段、资本收益手段。对于钱的处理是一种三角形思维，三个角分别是再生产、生产力工具、资本性收入。我们在讨论人生吗？如果你这样理解那就太粗浅了，当钱转化为资本的时候，我们站在国家层面思考资本的时候，同样也是这样一个三角形框架，也就是一个国家的财富是应该用于消费呢？还是生产力工具呢？还是提高资本性收益呢？三角如何平衡才能实现国家和社会的良性快速发展呢？如果失去平衡又会发生什么呢？

有些国家在特定时期为了使特定人群获得资本性收益，通过资产溢价，例如楼市或者股市的溢价，那么政府会向市场投放过多货币或者流动性，在这个过程中会形成一种价值观或者文化现象，鼓励人们在某方面过度消费，例如过度买楼，过度房地产消费导致拥有信贷资源的人，可以大量持有土地资源或者房产资源，从而获得巨大的资本性收益。由于极度的扭曲，资本大量进入到资产炒作或者资产溢价过程，从而影响了生产力发展的投入，也就是实体经济萎缩，而金融资本在膨胀，这个时候国家就出现问题，这和人的情况是一致的。母亲是最好的财政部部长，她不一定学过会计，但她一定学过出纳，会精打细算使一个家庭的财政收支平衡。
