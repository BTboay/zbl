货币¶

货币的发展状况到今天为止，已经远远超过马克思时代货币的特征，如果我们无法深刻的理解货币，那么我们就无法深刻的理解现代经济体系，也无法深刻理解《资本论》。从《资本论》中的货币论述延伸到最新的货币理论，了解内生货币和外生货币的内涵，最终我们要跳出教条式的教科书，站在社会层面理解货币的意义和作用。
