货币的两化¶

马克思和列宁对“钱”都是极度厌恶的，列宁准备用黄金做一个马桶。钱或者货币会发生异化，这是《资本论》之外马克思的论述。在异化理论中就是针对“钱”，钱本身不是毒药，而是拥有钱的人或者接触到钱的人发生了异化，因为钱带有特殊功能使人出现迷茫和错乱，这种现象是广谱性的。

货币的两化即货币资本化、货币商品化。商品化是指当你拥有钱后，你可以购买任何东西也就是使他人的任何东西商品化，甚至人格。资本化是指将权力等处理事物的能力作为换取货币收益的手段，腐败就是权力的资本化。资本主义在资本化问题上是有问题的，也就是过度资本化，当然这是资本主义的特征。划定两化的底线，对于国家和社会发展是必要的，对于个人的人生也是如此。钱的获取是有道德的，钱的去处也是有道德的，财富是需要在道德层面进行思考的，这里所讲的道德是社会化或社会性。就个人而言，那些不可以商品化、资本化的东西一定不能商品化和资本化，无论是你的身体、权利等等，不要轻易将其资本化从而获取超额的收益，这是人生戒律和底线，这样才不会犯低级错误，人生才能有意义，不要以为有学问的人、有地位的人就不犯低级错误，很多人是不会守人的商品化的底线和人的资本化的底线的。
