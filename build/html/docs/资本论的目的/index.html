资本论的目的

马克思为什么在19世纪60年代落笔《资本论》？马克思在经历了一系列的革命和运动后，发表了大量的政论文章，并且发表了共产主义宣言，这个时候马克思认为必须为无产阶级做一些理论上的准备了。无产阶级争取他们的权力需要一系列的理论支撑，片段的政论是不足够的，需要给出一个系统的解释，马克思写资本论的目的就是给无产阶级斗争提供理论依据，以及为无产阶级通过革命获得政权后建立一个无产阶级专政的国家，提供理论指导。
