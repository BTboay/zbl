二 难以避免的“新冷战”——美国为转移国内结构性矛盾将中国拖入“新冷战”

随着美国金融资本主导的全球化的发展，美元货币体系的不断深化，越来越多的国家被纳入到这个体系之中，实际上导致美国的内政往往不再局限于美国内部，特别是其金融政策影响着世界各国的发展，当美国因为其金融危机的爆发，各种救市政策并未考虑世界各国特别是那些严重依赖美元体系的发展中国家的利益，导致2008年后发展中国家纷纷爆发危机，世界各国之间的矛盾不断激化。美国事实上可以通过不同的金融政策，实现对世界各个国家的控制和影响，这种方式比过去军事力量介入的方式要隐蔽，并且可以实现军事力量无法实现的目标。

委内瑞拉在查韦斯时期，通过革命将石油资源收为国有，随着美国的救市政策推高了国际石油价格，委内瑞拉在查韦斯执政期间通过石油贸易得到大量收益，于是委内瑞拉在这一时期拉拢拉美各国，形成所谓反美阵营，改善国内平民窟状况，加大社会福利投入，得到了大量老百姓拥护。但到了2013年随着美国量化宽松结束，没有大量的货币流动性注入能源期货市场，石油价格开始下跌，并跌破了上涨前期的价格，巧合的是查韦斯也在此时去世，新上任的马杜罗直接遭遇了能源出口危机，收入大幅下降，国内开支骤减，委内瑞拉国内大量社会矛盾凸显，美国乘此加大软实力入侵策动颜色革命推翻马杜罗政权。虽然这不是美国有意的金融政策调整所带来的，但是可以看到美国金融政策可以实现过去军事手段或者冷战对抗手段而实现的效果，并且其过程更加隐秘，代价更小。

金融资本通过跨国公司和产业转移实现对世界人力，资源等剩余价值的占有，通过美元国际货币的地位实现对他国实体产业利益的获取，这种金融法西斯化是世界各国必然抵制的，伴随世界融入以美元为主的全球化进程，美国的内政，特别是金融相关的政策调整影响着世界各个国家，事实上对世界各国的国家安全产生威胁。中国作为美国金融资本全球化体系至关重要的一环，当产业转移带来的利益慢慢被金融资本带来的危害所超越时，中国试图摆脱美国的全球金融体系是理所当然的。奥巴马政府无法实现制造业回流，同时随着金融救市政策的实施，继续强化了美国金融资本力量，当特朗普政府上台后，其作为一个商人总统，直接将核心矛盾不加包装的展现在世界面前，作为接受产业转移最多最好的中国，当然成为美国转型的最大挑战和美国转嫁危机的最好场所，制裁中国让实体产业回流也是理所当然的。所以2008年后，随着中美双方的利益变化和各自的结构性调整，使得中美爆发直接性的冲突。

2017年特朗普政府上台，2018年开始和中国打贸易战。美国具有资源优势，加拿大，澳大利亚可以提供低价的原材料，美元作为国际货币其货币流通性对于商品贸易具有绝对诱惑性，美国是能源出口国，能源价格低，美国的税收也比中国低，除了人力资源成本，其他方面美国都具有比较优势，按理是可以有效吸引制造业回流的。

金融资本具有极强的排斥性，排斥任何实体产业，资本的逐利性决定了美国没有办法将金融资本的利润转移到实体产业中，无法将金融产业资本转变为实体产业资本，以美联储为首的美国金融机构并不由美国联邦政府控制，美联储是美国大型银行家共同成立的金融机构，其本身绝对服务于资本获利，所以美国的金融政策无法与实体产业融合，不能为实体产业发展提供支持。美国的资本力量借由美国资本主义自由市场制度，广泛控制了美国内部媒体和主流言论，美国金融的强大靠的是美元体系，而美元体系要靠美国军事力量支撑，金融资本全球获益后，部分收益回流军工体系，继续强化美国军事力量，这种循环致使金融和军工形成利益共同体，这股强大的政治力量是美国联邦政府无法对抗的。美国两党制导致两党随着其所代表的产业利益的矛盾深化而走向对立，形成激烈的党争，围绕党争形成美国社会各界的矛盾冲突，民主党代表以金融为主的产业利益，共和党代表以高新技术为主的实体产业利益，两党的矛盾在不断扩大，所以美国社会的矛盾冲突，其本质是金融资本和产业资本之间的矛盾，是获利的资本家和人民之间的矛盾，以党争和各种社会运动为主体表现出来。美国的政治体制，在创立初期就是服务于殖民者在美洲获利的，其核心从来不是人民大众，美国两党和政治议员代表不同产业集团利益，其本身就是和资本利益直接挂钩的，虽然由于社会矛盾爆发和国际地位下降，使得美国被迫走向产业改革，但其核心目的不是人民的生活而是资本的核心利益，所以这是其无法从高获利的金融产业转向实体产业的根本原因。美国需要一场社会主义革命并非胡言乱语，美国人民需要一场自下而上的社会革命，才能真正推翻金融资本的统治。

金融资本力量的一家独大和其在美国政治体制中的地位，导致美国无法完成转型，哪怕代表高新技术产业的共和党执政，其政治力量也无法改变现状，金融资本和产业资本之间不可调和的矛盾，这是美国必然走向衰落的根本依据，美国资本主义政治体制的原生逐利性是美国无法扭转实现改革的核心原因。如果以中国历史经验来看，美国已经处在因利益矛盾而产生的权力不断分割之中，资本主义自由市场体制其权力本就不够集中，伴随利益的分割，权力必然不断分割，中央政府不断失去控制力，所以有关美国分裂的论断是有据可循的。
