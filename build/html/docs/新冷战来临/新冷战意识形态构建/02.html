二 中国如何应对文明冲突论——客观归纳自身多元性，形成对话基础

我们需要构建一种对抗西方所提出的“文明冲突论”，”种族冲突论“的意识形态体系，所以我们提出中华民族文明的伟大复兴，而对西方世界来说绝对不允许这种东方文明复兴形成对西方文明的挑战，意识形态构建对于西方国家来讲是极其重要的，不管我们是否承认是否愿意，西方国家都一定要将我们纳入到它所构建的意识形态战场中的。尽管我们一再强调我们愿意合作不希望深化矛盾，但在西方国家霸权思维体系下，中国成为其单一霸权的直接威胁者时，西方国家是不允许我们合理存在的，所以我们必须为这场意识形态软实力战争做好准备。

西方国家的意识形态从殖民化时代开始，以文艺复兴为主要精神内核做出的一套全方位话语体系，其包装出的意识形态表达的内生性政治正确，对我们来说是很大的挑战。我们用什么作为我们话语构建的基础，我们怎么来解释世界呢？中国的两河和西方的两河，在地域范围上具有很大的差异，中国的两河跨度大，覆盖面积广，由于这种地理气候条件的不同，影响了不同地区不同民族的文化发展，所以中国文化自古就是多元的，具有包容性的特质，我们具有完全不同的独立的发展经验，而西方的小两河由于其地域狭窄，其文明具有典型的一元倾向，当西方的一元文化和东方的多元文化相碰撞时，一定发生严重冲突，我们只需将我们的多元性做出客观归纳，就可以形成话语体系和对话基础。
