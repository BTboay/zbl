三 美国面临危机——社会严重撕裂，出现非理性趋势

特朗普不惜煽动民粹主义，用法西斯化的民粹主义来促使美国实体产业复兴，因为其面对的是美国社会的严重撕裂。由于金融资本一家独大，产业资本不断衰败，而以金融为主的第三产业无法提供足够的工作岗位，越来越多的蓝领工人下岗，新增劳动力得不到就业机会，当社会基本需求无法得到根本保障时，最终导致美国社会的严重矛盾。随着产业结构的高度化，社会两级分化越来越大，贫富差距越来越大，伴随着美国社会的严重撕裂。

随着美国各种社会运动的兴起，却没有能够有效解决问题的方法，导致出现具有严重排斥性的法西斯民粹主义。在金融主导的全球化阶段，金融资本越是强大，越导致财富向少数人集中，越导致社会严重撕裂，最终导致社会非理性化，极端的民粹主义爆发。在这个新阶段所爆发的社会危机很难转化为我们所认识的，如共产主义革命时期的阶级政治，在这个几乎无解的复杂矛盾中，美国出现非理性趋势，这对我们而言，恰恰是一个理性解读世界的机遇，让我们冷静看待这个客体的客观演变过程，并以此为教训，这对我们未来发展极为重要。
