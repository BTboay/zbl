二 21世纪美国产业资本转移——美国国内产业资本流失，实体产业衰败

随着20世纪90年代末，中国开始搞大规模基础设施建设，到21世纪第一个10年结束，形成世界上基础设施最完善，质量最好的国家，这是一个全面的基础设施建设，并且中国14亿人口正在形成超大规模消费市场，形成大规模高素质劳动力。当美国在90年代产业资本不断退出，产业不断转移时，这些产业找到了中国这个最贴近市场，占有最多劳动力剩余的国家。随着美国各种实体产业向中国的转移，中美之间发生根本性改变，美国金融资本靠着这种跨国流动也形成大量获利机会，于是美国金融资本和中国实体资本在同一时期出现空前的高涨，而中美之间的产业互补关系，导致中美之间形成紧密战略关系，认为是中美10年蜜月期，并在世界上形成G2的国际关系。但是这一时期美国金融资本高涨，并不代表其收益来源于本土，随着金融资本的高涨，带来产业资本的更快速衰败，中国对更高质量产业化的需求和美国金融快速发展，导致短短10年，美国产业完全空心化，中国形成世界最完整工业制造体系和全产业链，并形成最大规模消费市场。

2008年美国金融海啸爆发，奥巴马政府通过三次量化宽松，增发3万6千亿美元，但是这些新增货币流动性，并不能流向美国的实体经济，因为美国此时的经济主动脉已经完全由金融资本组成，当金融危机爆发，美国政府只能帮助金融资本度过难关，这些新增货币都流向大型跨国企业，成为国际性投资，流向国际市场。金融资本对美国经济的主导，是美国金融资本主导全球化阶段的结构性矛盾之一。美国实体产业无法和金融资本一起获得良好的发展机会，反而因为资本的逐利性，逐渐退出了美国经济体系。新世纪第一个十年，美国金融业和以金融为主体的服务业占美国经济的比重不断攀升，在第一个十年结束，达到85%以上GDP占有率，成为世界产业结构高度化的典范。

产业资本的严重衰败，金融为主的第三产业的一家独大，这种美国经济的结构性矛盾，成为美国危机的根本原因。
