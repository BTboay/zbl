东马西马新马¶

马克思形成主义并开始落实于行动是在1864年，建立了国际工人联合会也就是第一国际，在这件事情上马克思与恩格斯是积极参与的，马克思提供了理论支撑和重要的指导。1876年第一国际解散，但是在欧美此时已经有16个国家建立了社会主义政党，此时马克思主义已经在欧美国家内发芽生根，结出了初步的果实。1889年7月14日国际社会主义者代表大会召开，为了纪念巴士底狱100周年，其名称叫社会党国际，我们称其为第二国际，这是在巴黎召开的一次会议，有22个国家383人参加，第二国际标志着马克思主义在组织和运动方面进入第二个高潮，其中一些历史性的人物出现了，例如瓦扬，拉法哥等。此时第二国际不仅仅是一个简单的组织和群众运动，他们推出了《劳工法案》《五一节案》。列宁在1914年8月14日宣布第二国际已死，第三国际万岁，1919年第三国际在莫斯科建立，我们称其为共产国际，负责同志是季尼科洛夫。第三国际在1943年5月15号解散，后来流亡的托洛斯基建立第四国际，但第四国际不被大家承认。三个国际代表着马克思主义走的三个阶段，其中第三国际代表着东马的正式产生。

从1943年到现在过去了80年，马克思主义在这个时间过程中在反反复复不断地演进和发展，其中东马的部分在中国经历了非常曲折的发展过程。第四国际应当由中国人来建立，虽然我们宣布我们不输出共产主义，我们不干涉他国内政，我们一直这样想也一直这样做，但形式的严峻程度往往超出我们的想象和预料。我们当前面对着西方逐渐联合对我们形成围剿之势，这个局势和当年对第三国际的围剿有类似之处，我们还是需要站在全球无产阶级的立场来思考这个问题。马克思主义一向是穿越种族阶级和国界的，正确的运用马克思主义既可以解释和解决历史遗留问题，也可以迎接新时代的到来，马克思主义的生命力在于当下可以解决严峻和复杂的社会问题，包括一些重大的国际问题，他可能使中国站在一个崭新的角度，开启一个崭新的时代，所以我们对第四国际有期待。
