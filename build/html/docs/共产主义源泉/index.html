共产主义源泉¶

18-19世纪的西欧有一群极其特殊的犹太人，这群特殊的犹太人对当代史和现代性起到了不可估量的影响，其中就有马克思。资本论是一部学术著作，但马克思主义却不完全是学术，共产主义更接近于一种宗教，因为它是一种信仰，而不是哲学推论的结果，这种信仰可以改变一个人，它可以使一个人纯粹高尚甚至高大起来。
