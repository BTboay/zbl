二 冷战后的中国——依靠有限能力维持基本盘

毛泽东主席在冷战中提出三个世界理论，点明了世界主要矛盾关系，当苏东解体，中国在推动货币化改革中遭遇严重危机时，美国没有将主要矛盾对象由苏东转向中国，当时的中国并未处在主要矛盾中。当苏东解体后，中国处于严重危机中，欧美达成的共识是中国崩溃论，且中国尚处于货币化改革时期，并未在金融领域有任何竞争能力，所以欧美不将中国作为主要竞争对手，并期待着中国的崩溃，美国可以继续收割中国。

90年代中国在跌跌撞撞，磕磕绊绊中度过，因为我们正处于货币化改革中，又遭遇外资撤离，只能靠自己有限的力量来维持基本盘。70-80年代中国只引进核心技术，核心设备，低端制造业和影响我国工业化进程的产业是严格控制的，但当90年代遭遇外资的撤离，不得不进行市场化改革，允许低端外资的进入。90年代中国沿海的开放，带来了低端制造业，并且打乱了上下游产业链关系，使中国进入到国际分工中，成为低端制造业的一环，为世界贡献一般消费品，形成三来一补的加工贸易型外向型经济。原本的产业体系被打破，上游设备制造失去下游市场，中小型国有企业倒闭，总计40万家，四千多万国企职工下岗，导致90年代各种社会乱像比比皆是。在这个阶段国家资本是非常薄弱的，国家财政也是十分薄弱的。

1997年东亚金融危机，本来中国和东亚国家形成了低端制造的贸易体系，随着东亚国家被金融危机打垮，中国也受到相当的冲击。整个90年代，我们只能是维持住国家不崩溃，社会基本稳定不动摇。
