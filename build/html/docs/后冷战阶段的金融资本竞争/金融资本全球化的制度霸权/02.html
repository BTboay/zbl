二 欧洲不顾美国反对发行欧元——金融资本全球竞争中欧元成为硬通货

苏东解体后，金融霸权应当属于美国，但没有哪一个国家和地区甘愿受美国的驱使，甘愿美国控制其国家主权和国家安全。1994年之后短短5年，欧元问世了，欧洲作为一个庞大的经济体，其也在苏东解体中，吃进了苏东的资产，同样导致欧洲资源市场扩大，产生的区内贸易超过了区外贸易，成为欧洲经济贸易的主体。所以欧洲出现区域货币，将区内贸易收益紧紧掌握在欧洲人自己手中，不被美元体系收割，是必然的，所以欧洲不顾美国反对1999年正式发行欧元，由于欧元主要服务于区内贸易，其使用效率大大高于已经虚拟化了的美元，所以欧元问世时币值高于美元。同期欧洲向世界派出了各种“说客”，在各个拥有贸易盈余的国家推广欧元。所以在金融资本阶段开展全球竞争的过程中，陡然出现一个欧元，同时人们开始讨论亚元，整个90年代中国纳入全球贸易体系中，由于我们接纳低端制造业，所以我们慢慢形成贸易盈余，开始大量储备美元，日本在战后大量引进美国产业转移，也拥有大量贸易盈余，而韩国在90年代后开始接受日本产业转移，也逐渐成为加工贸易外向型经济，东南亚的四小龙，四小虎也都是接受西方产业转移形成贸易盈余的国家，所以在东北亚地区形成收纳美元的美元湖，是一个大量储备美元的地区，并且东亚地区区内贸易也不断强化，所以对亚元的讨论也成为一种趋势。

欧元的形成和亚元的讨论，以及其背后所形成的区域经济结合，以及以区域贸易为核心的新地缘关系的重组，必然冲击美国一超独霸的地位，特别是对美元资本体系和国际金融地位的冲击。所以整个90年代和新世纪的头十年，尽管都是资本主义国家，但并不因为都是资本主义国家就不发生竞争，就不爆发战争，战争本就属于资本主义。

当欧元问世，很多国家开始接纳欧元作为主要结算货币，改用欧元进行贸易结算，其中包括朝鲜，伊拉克。朝鲜之所以没有被美国直接以军事手段灭国，主要在于朝鲜是贸易逆差国家，无法对国际金融资本市场产生冲击，并且朝鲜有中国和俄罗斯的战略保护，所以美国没有对朝鲜进行军事打击，而所有其他国家基本被美国以军事手段灭国。伊拉克在1999年和欧元发行同步宣布使用欧元结算，伊拉克在1999年是一个拥有年400亿石油贸易盈余的国家，他是贸易顺差国，当其改用欧元结算，意味着每年在国际金融市场抛售几百亿美元，而买进几百亿欧元，如果美国不介入，最终在国际金融市场会导致多米诺骨牌效应，其他国家资本也会加入抛售美元的行列，美元在国际金融市场的地位和信用就会被受到严重打击，最终影响美国通过美元在全球化进程中获利的基本经济模式，动摇美国根基。所以不管伊拉克有没有什么大杀伤性武器，借口一个人权问题直接干掉，萨达姆·侯赛因吊死，给世界看看。

资本主义国家，并不因为信奉一个主义，就没有竞争和冲突，整个新世纪的头10年，欧洲成为美国在金融主导全球化竞争过程中的主要竞争对手。
