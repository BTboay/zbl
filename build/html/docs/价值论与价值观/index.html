价值论与价值观¶

价值论是《资本论》最重要的一节。所有经济学，特别是宏观经济学的起点一定是价值论，广义上所有哲学研究的核心问题也是价值论。我们经常讲到“三断”，断是非，断大小，断远近，三断过了人生无碍。断是非是一个道德判断，断大小是一个价值判断，断远近是一个关系判断，貌似三断，其实他们的核心都是价值论。
