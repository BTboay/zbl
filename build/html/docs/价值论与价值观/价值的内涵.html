价值的内涵

马克思在写资本论的时候，对价值做出了明确的定义，”凝结在商品中的一般劳动“，这个定义是精确的，不过他定义了劳动创造了商品，所以商品凝结了一般劳动，这是一个经济学的定义，相对狭隘了。今天我们给出一个超过经济学范畴的定义，“价值是主观对客观意义的一种判断“，价值是一种判断，是唯心的，所以价值的判断和判断主体，以及他所处的环境是相关的。所以，万事万物皆有价值，只不过当它没有被主观发现的时候，这种价值不做表达，当它被主观发现甚至在市场进行交易的时候，就可能形成对该事物的价值共识。我们一般用“意义”而非价值，主观对客观的一般意义判断，就是马克思所说的使用价值概念，只有拥有使用价值才有意义。价值判断是唯心的，是主观的意识行为，所以价值就是相对的了，如果一个事物通过交易进行了兑现，那么其所表达的货币价值或者其他交换物价值，就称为交换价值，也就是我们通常定义的“价格”。

价值的内涵本质上是主观对客观，重要性或者意义的一种判断，一种分别。从资本论价值论这一章开始，我们就已经从自然人上升到社会人了，高等动物才有复杂的价值判断，而低等动物只是一种需求性判断，低等动物的需求主要是一种食物的需求，但在价值判断上，人类有时候比低等动物更愚蠢。价值观会被误导，将毫无价值的东西判断为有价值，而将有价值的东西判断为无价值。如果我们自己无法构建合理的价值论，那么会导致我们对价值误判，以至于用错误的价格进行交易。

当我们开始在人生中进行“三断”判断的时候，我们实际上起了“分别心”，当我们有了分别心，我们就变成了社会人，既脱离了自然属性也脱离了神性。当我们成为社会人的时候，我们希望我们自己是拥有主体性的社会人，需要用心来判断，“正心以中”的中是对价值的精确把握，合理性的把握，边际的把握。当我们明确了价值判断的合理性和边际，做到“正心以中”，我们就不会出卖我们的国家，出卖我们自己的人格，出卖自己的身体去获取那些并无生存意义的东西，我们也不会被错误的价值观所误导，在市场上进行不对等的交易。“正心以中”是哲学上让我们回归价值本源的一种方法，在经济学上的意义就是找到真正的价值平衡点，并对我们所处平衡点的相对位置做出判断，并指导我们完成投资的过程。如果我们再进一步，上升到佛学高度，价值判断就是龙树菩萨的中观，包含了价值讨论的第二层含义，即从有分别，准确分别到去分别和无分别的过程，我们心中懂得分别，但是到了必要的时候我们放下这些分别，我们不对客体做一般意义的简单判断，并以此来指导我们的取舍或者行为。佛学的意义是要让我们认识到这只是我们唯心的主观印象，在本质上是空的。中观的意思就是在空和有之间，好和坏之间，左和右之间，不垢和不净之间，当价值理解到这样一个境界的时候，我们可以超越价值而不被价值所绑架，可以做到予取予求。

一切的道德判断都是源于价值判断，一切审美的判断都是源于价值判断，一切的重要性判断都是源于价值判断，这是价值论的外延。不要小看了在社会上流行的所谓意识形态或者文化现象，他们的底子全部来自于价值论，这种价值论的外延，形成了人们或者社会的整体价值观。当用价值观去看人生的时候，就是人生观，当用价值观去看世界的时候，就是世界观。如果不能接触价值论，不能建立起自己对价值的判断的独立体系，我们的主观，我们的主体性就无从谈起，这时你所认为你正在判断的那个价值观，可能是强加给你的价值观，3岁看大，7岁看老，就是在说价值观生成的过程。价值论是我们解放的起点，我们基于主体性的发现，然后重新对客体意义做出判断，这是我的判断，并且是“正心以中”之后形成的判断，那我们才能超越社会广谱性的价值观，而不被带偏，这时我们才具有独立的高耸的人格，有悠远的目光。对事对人的价值判断决定了整个社会的状态，在经过整理之后，价值观上升为一种宗教的体系，哲学的体系，文化的体系，深刻的影响了每一个人的行为逻辑，甚至构成普通人的悲欢离合。“观”就是意识，当集体的意识形成了，就是意识形态，而这个意识形态将塑造这个社会的伦理法理。

马克思在写资本论的初期，为什么要先把价值，“使用价值”，“交换价值”，“相对价值”这样的价值体系说清楚呢？是因为马克思准备剖析，劳动者劳动所形成的价值，其中一部分被不合理窃取了，也就是剖析剩余价值。马克思的哲学一般功力是深厚的，他采用的方法也是正确的，但是事物发展是迅速的，当今的时代比马克思所处的时代要复杂很多了。马克思时代的资本化过程和当下的资本化过程是有巨大差异的，人类的活动和生产已经相当复杂了。当今时代是一个资本过剩危机的时代，但是我们依然可以将其看作是生产过剩的危机，因为人们生产的不再是一般商品，而是货币。当货币成为一种商品的时候，那么整个价值的判断是要发生改变的。
