五 绿水青山就是金山银山——以国内大循环为主体，完成生态经济价值化

2017年乡村振兴战略形成后，将会构成一个以国内大循环为主体的发展战略基础，乡村振兴是其主要载体。国家实现主权可控的生态资源价值化，将生态资源变为可吸纳货币的主要领域，通过主权货币，货币化资源型资产成为生产力要素。绿水青山就是金山银山，在概念上扩展了生产力要素，这是一个非常重大的调整，山水田林湖草沙是一个完整的生命体，应该综合规划系统开发，不能用粗暴的市场方式，切分开然后七零八碎的卖掉，将巨大的生态资源转换为可交易资产，是我们形成国内大循环的基础，将人民币重新锚定到这些国家可控的资源型资产上，这是国内经济大循环的重要载体。只要这个领域做的好，我们将不再依赖外汇储备形成国内货币增发条件，改为锚定国内国家可控的巨大资源型资产上，以人民币作为金融投资在国内形成收益闭环。这是未来我们应对全球化危机的重要压舱石。
