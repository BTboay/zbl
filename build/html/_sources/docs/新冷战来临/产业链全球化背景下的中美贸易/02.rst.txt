二 从Chimerica到G2——中美之间呈现战略互补关系
=================================

如果用产业资本阶段的指标来衡量，中国早在10年前，就成为世界产业资本第一大国，只不过当使用GDP为经济指标的时候，我们排在第二的位置。

美国的产业资本外移，占有转出地的剩余价值，实现20%投资回报率回流美国，而中国随着美国产业资本的进入，货币在不断升值，产业能力在不断增强。随着中国实体产业的发展，商品在国际市场的流通，中国形成大量贸易盈余，中国货币发行不得不与美元挂钩，每进入1美元外汇，增发7-8块钱的人民币。所以在新世纪的第一个十年，随着外资的进入，以及外资买全球卖全球，中国形成大量贸易盈余，推动本币的大规模增发，中国客观上成为产业第一大国，贸易第一大国，金融总量第一大国。大部分产业资本在中国形成的收益，是美国跨国公司创造的，形成的是中国大量的外汇储备，但中国的外汇储备无法在本国使用，而我们与美国并非平等贸易关系，美国在核心技术，军事装备等中国急需的领域是不向我们开放的，我们没有大量使用美元的条件，不得不将大量美元返回到美国国债市场，至少可以获得一点投资回报。虽然中国在新世纪的快速发展，工业化的快速形成，是依靠美国产业资本的转移而实现的，但中国客观上成为美国金融资本全球获益的一环，我们不但向美国输出大量廉价消费品，并且形成的贸易收益受美国跨国资本影响，并且由于不对等贸易，不得不将外汇储备投入美国国债市场，帮助美国实现金融获利的闭环，所以当进入21世纪后10年，中国本国产业资本逐渐强大，与产业配套的相关技术设备逐渐国有化，我们的产业发展不再主要依赖美国产业资本和产业转移时，美国产业资本带来的好处渐渐小于美国金融资本获利的坏处时，中国开始从美国的金融资本全球获益体系中独立出来，从而导致中美在21世纪第二个10年出现利益冲突，并逐渐发展为两国在各个领域的全面竞争。

但在21世纪头10年，中美紧密的贸易关系，形成政治新关系Chimerica中美国，进一步美国人提出中美共治G2。
