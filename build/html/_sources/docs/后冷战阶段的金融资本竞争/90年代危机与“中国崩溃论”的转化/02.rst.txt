二 “中国崩溃论”出现——中国的“三年大萧条”
=================================

伴随着美西方对中国的硬拖钩和苏东的解体，提出“中国崩溃论”，中国本来就在危机之中，又遭遇到西方硬拖钩，在工业化高涨的时候，资本撤离，没有资本工业化无法发展，我们先于苏东开始改革，先于苏东遭遇严重危机，苏东从89年开始出现危机，中国在88年就开始严重通胀。中国不同于苏东的地方在于，我们没有开放资本市场，没有开放货币市场，对内的市场化改造并没有与世界资本市场联通，所以，当出现种种市场化问题的时候，中国央行通过大量增发货币，来对冲交易市场，通过完全放开票证供应，通过市场交易慢慢形成稳定的市场体系，虽然这个过程的冲击也是巨大的，但并未被国际投机资本扰乱市场秩序，收割中国实体资产，并且中国的基本生活必需品可以自给自足，从而并未发展为社会的极端分裂，导致国家覆灭，但这不代表我们轻松的度过了改革带来的危机，此时的宏观经济正处于严重危机之下，只是相比于苏联东欧，我们没有发展到国家解体崩溃的地步。

中国政府通过向自己的货币体系进行赋权，通过大量增发货币，替代撤走的外资，不外乎是资本投资，外资不干内资干，中国能够稳定住经济不跨，很大程度上是因为我们拥有自主的货币体系，当92年完全放弃票证供应，使用人民币来活化交易市场时，从这时人民币才正式成为货币，所以1992年称为人民币的货币元年。国家财政增发国债，用国债增大政府开支能力，央行增发货币，财政增发国债，用增发的货币购买国债，抬高国债资本价值，提升国家开支能力，这样的作法和美国通过回流资本推高美国国债是一样的，只不过美国的国债被全世界买走，中国的国债只能对内发行，国内工资快速提升，国债和货币大量发行，通过资本信用扩张来回避外资撤离所形成的危机。但当时中国经济是不景气的，是萧条的，随着货币的大规模增发，中国迅速进入通胀，89-91年严重通缩，92年进入复苏，93年进入严重通胀，通胀导致新发行的货币无法完全覆盖财政发行的国债，财政开始透支银行资本金，1994-1995年财政将银行资本金完全吃光，并透支了几十亿人民币，中国宏观经济进入严重危机阶段。95年中国开始进入宏观调控，97年宣布宏观调控软着陆成功，实际上中国在这个阶段问题更加严重，因为宏观调控的结果是紧缩，央行通过利率调整回收市场上的资金，又导致本来不景气的实体产业，遭遇资金流和资本投资困难，实体产业更加衰败。

92-93年当面临外资撤离时，为了吸引外资，沿海城市开放低端制造业市场，原来严格控制不让进来的高消耗，高污染的低端产业，此时全部进入中国。邓小平当时说，你只要开开窗户难免苍蝇蚊子要飞进来，大量低端外资进入，将沿海城市变为了靠外部设备来形成生产能力的“三来一补”【35】发展模式，叫做加工外贸型的外向型经济，其导致的结果是，国内上游装备制造业失去了下游市场，所以当90年代中期，遭遇通胀后再紧缩，本来就严重不景气的制造业，又失去了下游市场，纷纷走向倒闭，从90年代中期开始关停并转，抓大放小，中小型国有企业被放弃，职工大规模下岗。

注
=================================
【35】三来一补：来料加工，来样加工，来件配装以及补偿贸易的总称
