二 从新农村建设到乡村振兴——“三农”投资是中国应对全球化挑战的压舱石
===================================

2005年前是县一级政府承担“三农”责任，2005年后“三农”列入国家战略，中央政府承担了“三农”投资的主要责任，“三农”投资从2005年开始不断增长，五年后2010年三农开支成为国家财政最大开支。所呈现出的主要变化就是农村基础设施，水，电，路，气，宽带的全覆盖，为农村产业发展提供了基础，农村得到极大程度发展。到2012年再一次政府换届，习总书记在原有基础上，提出乡村振兴，其内容和上一阶段是几乎一致的，新农村建设将五通发展到行政村一级，乡村振兴将五通发展到自然村一级。国家对三农的持续性投资，实现了城乡差别再平衡，三大差别已经实现两大差别的再平衡了。

全面建成小康社会实现共同富裕，扶贫攻坚开始成为国家发展的主要方向，其目的就在于实现第三个贫富差别再平衡，新型的中国式国家主义因三大差别再平衡，而呼之欲出。这个举国体制不为别的，而是为了实现全面小康。以农业生产为基础的三农，其由于产业基本特性，无法有效实现收益增长，甚至呈现负增长，所以三农能否实现小康是全国实现小康的基本前提，所以三农开支才成为国家财政最大开支。

精准扶贫在国际上几乎没有成功经验，因为大部分发展中国家没有实现土地均分，农民没有土地，只能靠买体力为生，如果没有足够的产业发展提供工作岗位，那么农民只能贫困，其资产是绝对的零，一般将其称为制度贫困。而以中国为中心的东亚，东南亚国家都实现了土地均分，所以绝对意义的财产归零造成的贫困是不存在的，特别是中，日，韩三国，包括朝鲜。西方国家以基金会为基础做出针对性扶贫无法逆转，因为其无法提供房子，土地这样的资产，也不能扶持产业发展提供工作岗位。中国可以将产业扶持和扶贫相结合，一石二鸟实现精准扶贫。

美国三大产业中，第三产业占87%，第一产业1.2%，第二产业11%，这是一个高度结构化的经济结构，而中国随着国家投资拉动的区域发展，形成了庞大的金融资本，庞大的实体产业资本和健康发展的大规模农业产业，我们是全面的并且相对平衡的。随着基础设施的不断完善，使得农村具备了将资源性资产转变为可交易的实体资产的条件，这是国家稳定的基础，是我们应对国际威胁的压舱石。

中国的发展道路是人类在资本主义历史阶段前所未有的综合性发展道路，伴随着中华民族伟大复兴的征程，中国正在引领全人类探索新的发展道路，其完全不同于西方式的工业加城市化的现代化发展道路。中国应对美国发起的新冷战的基本方法，不是跳进美国所构建好的游戏框架中，而是坚持自己的发展道路和对未来的认知眼光，走好自己的路就是最佳的破解之法。
