发展中国家的选择和命运
=================================

发展中国家普遍接受西方式现代化模式，追求工业化加城市化，当发达国家开始转移低端制造业，发展中国家自然接受转出的制造业来发展本国工业。制造业转出不是赠送，发展中国家没有与之对应的交换物，一般以负债形式接受转移。发展中国家都是从殖民地独立出来的，没有任何工业化经验，其不得不仿照发达国家国家调整管理体系，教育体系等来支持发达国家转移的产业，其带来的是全面西化，所以产业转移带来的不仅是产业本身，也带来了西方的制度体系，文化理念，教育模式，其背后所蕴含的是发达国家的软实力，硬实力，巧实力。随着发达国家产业转移，发达国家向金融资本阶段转变，而发展中国家对外贸易，不得不进入以美元为核心的金融全球化，不得不接纳美元等金融体系的掠夺，而发展中国家普遍没有金融交易的经验，普遍依赖西方国家的金融机构代理，使得发展中国家不但付出了接受转移产业所转移的危机，又受到金融资本的掠夺，并且其主权和文化体系遭到入侵。而随着产业转移，其教育和文化同样接受西化改造，于是发展中国家在面对种种矛盾和剥削时，用修饰过的西方话语来描述自己，更加丧失了在话语上的主动权，迷失在了所谓自由，民主，人权等毫无意义的辩论中。
