二 80年代后的美国发展
=================================

从80年代开始美国进入以货币交易来推动虚拟市场发展的阶段，80-90年代短短10年，美国开发出2000多种虚拟衍生品交易，把各种资产打包（package）变成一种虚拟交易，美国大量的虚拟交易所形成的交易量是实体经济的成百上千倍，这种没有成本可以快速获利的经济体系迅速将实体经济排斥，实体经济渐次退出了美国的经济发展过程。为了统计在虚拟交易中获得的收益，即交易过程的增加值，美国开始使用GDP（交易增加值）来统计经济增长，世界进入GDP时代。

资本市场交易周期极短，交易增加值极高，所以当美国改变经济发展模式，同时改变经济统计方式后，美国经济在世界上脱颖而出，虽然金融交易不产生实际“价值”。美国成为美元世界的主导，及西方国家的主导，而跨国公司对转移产业的控制，以及美元对主要大宗商品的控制，使得美国成为多数发展中国家的主导，从此，美国从美苏双寡头地缘战略博弈中显示出明显上升趋势。

美国和西方发达国家的崛起，是在产业转移后，逐渐向金融资本阶段发展，通过虚拟资本获利而实现的，是西方带领世界走向金融主导的全球化的时代。美国在里根总统时代和英国在撒切尔夫人时代，大力推进经济结构向金融转型，英美日三方紧密配合，联手向金融化方向发展。美元增发，英镑和日元通过购买新增发的美元，从而维持美元币值，英日的货币政策和美元是紧密相关的。

70年代开始中美邦交正常化，美国和西方国家对中国开放民用市场，事实上将中国纳入到了美国以及西方的世界分工体系，开拓了中国这个大市场。中国从70年代开始引进技术，设备和资本，中国进入高负债阶段，发展中国家普遍变成负债国，发达国家变为债权国。
