马克思主义的哲学源泉
=================================

要讲清楚马克思哲学的思想源泉，我们不得不重温这400年的世界思想史。我们需要理解这400年的欧洲思想家们是如何思考的，欧洲是如何基于这些思考而完成从工业化到现代化的，他们经历了怎样的心路历程。思想史是具有时间长度和空间广度的，这一节我们将其浓缩到四个人，通过这四个人徐徐展开，分别是“笛卡尔”，“卢梭”，“康德“，”黑格尔“，两个法国哲学家，两个德国哲学家，他们构成了整个欧洲思想史的脉络。他们思考的问题在当下的中国依然是问题，而且大部分的问题特别是关于现代性和后现代的问题实际上一个都没解决，这是一个沉重的话题，这一章借讨论欧洲哲学史和思想史，也检讨一下我们在哪里出了问题。站在历史的长河中，很多事情具有自然的原因，但同时也有人的原因，东方哲学家并不比欧洲哲学家弱，只是东方人一直没有能够从沉重的旧有的哲学历史包袱中走出来，而欧洲哲学家破解了中世纪欧洲的经验哲学。中国思想家和哲学家一直无法走出儒家思想的经验哲学，当今中国在诸多问题上的思考任然是儒家经验哲学的现代翻版。

哲学是关于智慧的学问，那么什么是智慧呢？我们所处的世界可以概括为四种关系，人与自己的关系，人与社会的关系，人与自然的关系，人与神的关系，说清楚这些关系就是哲学。在漫长的中国哲学史上，中国的哲学家早就会描述这四种关系，中国的哲学家比古希腊和古罗马的哲学家更早的完成了在各种复杂关系上的称述和定义，甚至包括在自然科学方面的称述和定义，我们整个理论的成型和成熟是在汉，在成熟之后对旧体系的破解，王阳明算是一次努力，但他的奴隶并没有实现对中国式儒家经验哲学的有效破解，以至于在明末清初这样一个复杂的历史时期，柔和了复杂的民族矛盾，种族矛盾，阶级矛盾，当时的思想家们没有给出一个完整的答案，甚至鸦片战争洋人用枪炮打开中国大门的时候，中国的思想家们也没有给出答案。真正有价值的思考要到五四运动的时候才开始，但是那个思考在今天看来依然是缺乏哲学高度和历史深度的，以至于百年后的今天中国人又开始进行复杂的思考，来解释今天中国的合理性。

黑格尔所说“存在即合理”，今天的中国成功的崛起了，那么我们如何站在哲学的高度解释我们的道路，我们如何分析和判断我们道路中的问题并且以高瞻远瞩的方法来解析和看待我们的未来，我们的未来是79年要走的路还是300年应该走的路。欧洲的思想家和哲学家是值得敬仰的，他们是了不起的一代人，是真正的精神贵族，构建了一个时代的文明。
