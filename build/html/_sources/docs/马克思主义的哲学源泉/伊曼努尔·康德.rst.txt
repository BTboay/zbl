伊曼努尔·康德
=================================

德国哲学家，出生于1724年在1804年逝世，他在卢梭之后在马克思之前。我们认识康德是从他的三个批判开始的，纯粹理性批判，实践理性批判，批判力批判，康德的三个批判构建了康德的哲学体系。康德的生活是很有规律的，他甚至没有离开过他生活的小镇，他每天下午三点必须去公园散步，唯一一次打破这个时间，是他在读卢梭的《爱弥儿》时被震撼到了，他想继续读完，所以没有去散步。

世界公民和世界联邦的提法是康德提出的，可以想象他有多么宏大的视野，他在他的时代就提出了不干涉内政的主权国家概念。康德和黑格尔具有继承性，他们作为德国人开始思考德意志民主和德意志国家的关系，黑格尔是德意志国家精神的接生婆，他从哲学高度奠定了德意志民族国家的基础。康德的贡献在于他将哲学的方法做了历史性的思考和批判，他认为人应该对自然立法，事物的特性与观察者有关，这貌似有点主观唯心主义的意味，但实际上他打破了旧有哲学思考的角度，这是一种哥白尼式的革命，他将“我者”和“他者”以及“第三方”的关系做了调整。他将英国的经验主义和法国的理性主义做了调和，这个调和构成了康德独特的哲学体系。在经验上进行哲学高度的概括和归纳就变成了理性主义，理性主义是通过逻辑构建的整套体系，通过这套逻辑体系对未知事物做出判断。当经验主义和理性主义相结合的时候，才是一种符合实际的哲学思考，所以在某种意义上康德也正在挣脱逻辑主义的枷锁，为德意志民族国家寻找出路。

经验主义和理性主义的融合是一个激烈的碰撞过程，毛泽东在实践论中对经验主义和理性主义有着精彩的论述，这也构成历史唯物主义和辩证唯物主义的基本工具和方法。
