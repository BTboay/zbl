二  殖民主义全球化的演变——从航海大发现到奴隶制三角贸易
=================================

早期东西方贸易逆差对殖民主义全球化起到重要作用，在这个演变过程中为了维持海上航路，就需要各种配套科学技术，当欧洲需要在美洲开发银矿，也需要各种技术跟进。于是需求带动发展，天文学，地理学，气候学，海洋学等各种知识得到快速发展【7】，自然科学崛起是在这一阶段出于客观需求而形成。

对外殖民，获取到大量原材物料，但是因为侵略行为导致本地原住民锐减，为了补足劳动力来开发这些原材料或者形成大规模种植园，最终入侵非洲，从非洲原住民社会获取劳动力。形成了三角贸易，美洲资源流向欧洲，欧洲人流向非洲掳掠劳动力，劳动力流向美洲开发资源，这是一个殖民主义的原材料掳掠过程。

早期海上霸主西班牙，由于殖民掠夺快速强大，但是欧洲还没有可以与之抗衡的其他力量，外部刺激不足，大量掠夺的资源被用于西班牙统治者，也就是西班牙的天主教会的挥霍，修建装饰教堂等等。所以西班牙没有进入工业化的推动力量，而英国是后起之秀，为了打败西班牙的无敌舰队，需要更高的技术支撑，更强大的生产力，于是带着这种外部刺激，最终形成工业化。

英国作为工业化进步的强权，将掠夺的资源投入工业化生产，最终借助工业化带来的优势打败其他海上强国（荷兰，西班牙），占有世界大部分原材料产地，对世界做原材料掠夺，以这种方式完成了资本的原始积累。最终形成以工业生产为主体的产业资本全球扩张。

产业资本有一个重要的内在本质，产业是在国化的，产业资本也是在国化的。

注
=================================
【7】现代自然科学发展主要从天文学，地理学等开始，其主要服务于远洋航行需求，其后催生出几何学，物理学，最终抽象出统一建模工具数学，自然科学发展史和欧洲对外殖民史是直接对应的
