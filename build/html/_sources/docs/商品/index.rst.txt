商品
============================================

当我们理解商品化之后，我们自然而然进入了资本化的过程中，这是《资本论》第一卷的核心。商品的概念我们从小就接触，但越是看似简单的东西越复杂，我们真的懂什么是商品吗？懂什么是商品化吗？懂什么是商品定价吗？商品定价的逻辑又是什么呢？这一系列的问题下来，就可以看到马克思了不起的地方，他总能抓住根本要点，再层层剖析。

作为个人投资者对经济学的理解可以分为三大块

1.创造价值理论：它是人生观、世界观，如果不懂得创造价值，人是没有意义的。

2.商品定价理论：只有知道事物的价值才能进行投资，懂资本定价逻辑才能运作资产。

3.价值投资理论：投资的价值确定才能确保投资的收益。

商品定价是其中的核心，定价过程中有其道德属性、合理性因素，我们应当如何看待凝结在商品中的各种要素，最终形成一种定价逻辑。马克思在不合理定价中发现了剩余价值，当发现问题后，我们需要开始对制度进行批判，对人生以及诸多问题进行崭新的思考和认识。
