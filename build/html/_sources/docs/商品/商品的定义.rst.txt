商品的定义
============================================

似乎每一个人都知道商品的本质或者定义，为什么还需要谈商品的定义呢？因为我们认为马克思和恩格斯对商品的定义，是一个狭义的商品定义。我们给出的商品定义是“用于交易的一切事物皆是商品”。从哲学上看，商品是人类的属性，是一种文明行为，动物是没有商品交换的，良好的商品交易过程可以取代战争，所以是文明的。商品是价值、使用价值、交换价值的统一体，每个人眼中对商品的评价是基于自己的不同需求，形成了商品的价值，这是唯心主义的。商品建基于人类需求，商品生产是一种供给的生产行为。

马克思在讨论商品时有一句很重的话“资本主义生产方式占统治地位的社会财富，表现为庞大的商品堆积”，这不仅仅是物的堆积，也逐渐形成资本主义社会的一种审美，这种审美的畸形发展的产物就是“拜物教”。我们通常理解的商品价值往往是交换价值，我们不能精算每一种商品凝结的劳动，更难以了解商品的具体使用价值。当交换价值严重背离商品的本来价值，可能是稀缺性造成，可能是垄断造成，就会出现某种暴利，这个时候我们需要思考制度设计问题。
