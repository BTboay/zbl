资本的分类
============================================

在今天的西方经济学体系中，资本是不分类的，而在过去苏联的教科书中，资本进行了详细的划分。通常我们把资本分类如下：

1. 农业资本
2. 工业资本
3. 商业资本
4. 金融资本

我们应当对资本有一个明确的分类，因为资本在不同产业中的资本利得的差距是巨大的，资本的构成是不一样的。通常农业资本都是以自有资金为主，工业资本或者产业资本就无法完全以自有资本为主了，自有资本此时只能是一个影子，当进入商业资本时，自由资本的占比就更低了，其主要以融资或信用征集为主体，而金融资本就完全没有自有资本了，不同类别的资本决定了不同的资本利得体系和资本结构。
