劳动
============================================

《资本论》立基于四个东西，价值论、货币、劳动、商品，这是《资本论》的四块基石。马克思这样的大思想家，他做学问的基点是非常扎实的，真正的大家都是从最朴素、最细微的概念入手，挖掘出别人看不见的东西，最终形成对历史内在规律的深刻把握。我们都在劳动，但我们对自己的劳动、劳动与资方的关系、劳动与政府的关系、劳动与社会的关系没有清醒的认识，以至于我们无法清晰的表达。

在很大程度上理解了劳动，才能理解人与人、人与社会、人与资方、人与政府的关系，甚至懂了劳动，才懂得怎样教育自己的子女。因为劳动作为一种狭义的劳动，作为一种商品的劳动、作为一种可交易品的劳动，其实是需要进行某种养成的，它包括了教育、学习、训练诸多因素。我们需要超越马克思《资本论》中劳动的侠义范畴，站在不同角度深刻理解劳动的本质。
